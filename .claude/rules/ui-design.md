# UI Design Rules

- Dark blue-green gradient background — maintain this aesthetic in all views
- Card-based layout with semi-transparent rounded rectangles (opacity ~0.15-0.25)
- Clickable cards use hover effects and chevron indicators for discoverability
- Main popover: 420x640, detail panels: 370x560
- Use SF Symbols for icons — match existing icon weight and style
- Temperature displays support both °F and °C via the global toggle
- Footer pattern: quit button (left), app name (center), settings/toggle (right)
